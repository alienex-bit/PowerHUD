# Oxygen Bar Samples (ASCII)

These are ASCII-safe samples for the oxygen bar line so they render in all fonts.

## Option A: Thin track + percent
Oxygen: O2 72% |========----| Safe
Oxygen: O2 41% |====--------| Caution
Oxygen: O2 22% |==----------| Low
Oxygen: O2 08% |------------| CRITICAL
